# **App Name**: Hajaty Hub

## Core Features:

- User Authentication: Secure user login and signup using Firebase Authentication.
- Service Catalog: Browse and discover SMM and digital services.
- Order Management: Place and track orders for various services.
- Ad Campaign Center: Create and manage ad campaigns across multiple platforms.
- Affiliate MLM System: Unique Referral code to other users to generate a multilevel marketing based referral system.
- AI Assistant: Provide real-time support and guidance via a floating AI tool.
- Rank and Discount: Users gain new ranks with higher levels of discounts based on the amount of total spending on the site.

## Style Guidelines:

- Primary color: Saturated blue (#3498DB) to reflect reliability and trustworthiness.
- Background color: Light grayish-blue (#ECF0F1). A desaturated tone of the primary hue that gives the application a smooth look.
- Accent color: A purple (#9B59B6) will be used as an analogous tone with the blue tone, yet set apart via a high saturation and distinct brightness
- Font pairing: 'Poppins' (sans-serif) for headlines and 'PT Sans' (sans-serif) for body text, to ensure clear readability and a modern feel.
- Lucide React icons for a clean and consistent look.
- Arabic RTL (Right-to-Left) support throughout the application.
- Subtle animations to enhance user experience.
